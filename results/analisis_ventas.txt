**Análisis de Ventas por Tipo de Producto**

En el análisis de las ventas por tipo de producto, se observa lo siguiente:

- **Lácteos** es la categoría con el mayor número de ventas, alcanzando un total de 150 unidades. Esto sugiere que los productos lácteos son los más populares o tienen una mayor demanda entre los consumidores.
- **Frutas** es la categoría con el menor número de ventas, con solo 60 unidades vendidas. Esto podría indicar una menor preferencia por las frutas o una posible necesidad de revisar la estrategia de ventas para esta categoría.
- Las otras categorías se distribuyen de la siguiente manera: **Bebidas** con 120 unidades, **Panadería** con 90 unidades, y **Snacks** con 80 unidades.

**Conclusiones:**

1. Los productos lácteos son los más vendidos, lo que podría ser una oportunidad para expandir la oferta en esta categoría o realizar promociones para mantener el interés del consumidor.
2. Las frutas, al tener el menor número de ventas, podrían beneficiarse de estrategias de marketing o promociones para aumentar su atractivo.
3. Las bebidas también muestran un buen desempeño en ventas, lo que podría ser una categoría a potenciar.

**Gráfico de Barras:**

Para visualizar mejor estos datos, se puede generar un gráfico de barras que muestre las cantidades vendidas por cada categoría. Esto ayudará a identificar visualmente las diferencias en las ventas entre las distintas categorías de productos.

```python
import matplotlib.pyplot as plt

# Datos de ventas
categorias = ['Bebidas', 'Snacks', 'Lácteos', 'Frutas', 'Panadería']
ventas = [120, 80, 150, 60, 90]

# Crear el gráfico de barras
plt.figure(figsize=(10, 6))
plt.bar(categorias, ventas, color=['blue', 'orange', 'green', 'red', 'purple'])

# Añadir títulos y etiquetas
plt.title('Ventas por Categoría de Producto')
plt.xlabel('Categoría')
plt.ylabel('Cantidad Vendida')

# Mostrar el gráfico
plt.show()
```

Este gráfico de barras proporcionará una representación visual clara de las ventas por categoría, facilitando la comparación y el análisis de los datos.
