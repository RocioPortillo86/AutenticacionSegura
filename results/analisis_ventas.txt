**Análisis de Ventas por Tipo de Producto**

En el análisis de las ventas por tipo de producto, se observa lo siguiente:

- **Lácteos** es la categoría con el mayor número de ventas, alcanzando un total de 150 unidades. Esto sugiere una alta demanda de productos lácteos entre los consumidores.
- **Bebidas** sigue en segundo lugar con 120 unidades vendidas, indicando también una fuerte preferencia por este tipo de productos.
- **Panadería** ocupa el tercer lugar con 90 unidades, mostrando una demanda moderada.
- **Snacks** se encuentra en el cuarto lugar con 80 unidades vendidas.
- **Frutas** es la categoría con menos ventas, totalizando 60 unidades, lo que podría indicar una menor preferencia o necesidad de promoción adicional para aumentar su demanda.

**Conclusiones:**

- La categoría de **Lácteos** es la más vendida, lo que podría ser una oportunidad para ampliar la oferta de productos en esta categoría o realizar promociones para mantener el interés del consumidor.
- **Frutas** es la categoría con menos ventas, lo que sugiere que podría beneficiarse de estrategias de marketing o promociones para aumentar su atractivo y ventas.

**Gráfico de Barras:**

Para visualizar estos datos, se puede generar un gráfico de barras que muestre las cantidades vendidas por categoría. Esto ayudará a identificar visualmente las diferencias en las ventas entre las distintas categorías de productos.

```python
import matplotlib.pyplot as plt

# Datos de ventas
categorias = ['Bebidas', 'Snacks', 'Lácteos', 'Frutas', 'Panadería']
ventas = [120, 80, 150, 60, 90]

# Crear el gráfico de barras
plt.figure(figsize=(10, 6))
plt.bar(categorias, ventas, color=['blue', 'orange', 'green', 'red', 'purple'])
plt.title('Ventas por Categoría de Producto')
plt.xlabel('Categoría')
plt.ylabel('Cantidad Vendida')
plt.show()
```

Este gráfico de barras proporcionará una representación visual clara de las ventas por categoría, facilitando la identificación de las tendencias y áreas de oportunidad.
